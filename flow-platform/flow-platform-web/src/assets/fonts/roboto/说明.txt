(1) 字体文件：Roboto
从 Google Fonts 下载 Roboto 字体。
放置在 assets/fonts/roboto 目录下。
在 global.css 中引入：
@font-face {
  font-family: 'Roboto';
  src: url('./assets/fonts/roboto/Roboto-Regular.ttf') format('truetype');
  font-weight: normal;
  font-style: normal;
}

(2) 图标字体：Font Awesome
使用 Font Awesome 图标字体。
在 index.html 中引入：
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/5.15.4/css/all.min.css">
